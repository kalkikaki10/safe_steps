Safe Steps – AI Assistive Navigation System

Safe Steps is a web-based assistive technology platform designed to help visually impaired and blind individuals navigate their surroundings safely and independently. The project combines artificial intelligence, computer vision, voice interaction, and real-time navigation to create an accessible digital assistant that improves mobility and safety.

The system provides multiple intelligent features that allow users to interact with the environment through voice feedback and simple controls. The main interface presents different modules such as live object detection, navigation assistance, document reading, and an AI assistant, allowing users to easily access each feature. 

One of the key features of Safe Steps is Live Object Detection, which uses the device camera along with AI models to detect objects such as people, chairs, and vehicles in real time. Detected objects are highlighted on the screen and voice feedback informs the user about obstacles nearby. 

The Navigation Aid feature allows users to reach a destination using voice commands. The system captures the user’s spoken destination and opens walking directions through Google Maps, enabling independent navigation. 

Safe Steps also includes a Document Reader, which uses Optical Character Recognition (OCR) technology to scan printed text and read it aloud. This allows visually impaired users to access information from documents, signs, or printed materials easily. 

Another feature is the AI Assistant, which allows users to ask questions using voice or text input. The assistant retrieves information from multiple online sources and responds with voice output for accessibility.

For safety, Safe Steps provides an SOS emergency system. When activated, it sends the user’s live location to an emergency contact through WhatsApp and can also initiate a direct phone call.

The platform also supports voice commands, enabling users to control features such as object detection, navigation, and document reading through speech recognition. 

User preferences such as emergency contacts, voice speed, and accessibility options can be managed through the settings page.

Overall, Safe Steps demonstrates how modern web technologies, artificial intelligence, and voice-based interaction can be integrated to build a practical assistive solution that improves independence, safety, and confidence for visually impaired individuals.
